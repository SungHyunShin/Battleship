# Battleship placement
Doesn't have any functional playing, only placing.
## To run
### Server
`npm install`
`npm start`
Runs on port 3001
### Frontend
`cd client`
`npm install`
`npm start`
[http://localhost.](http://localhost:3000/)

## Current State
Frontend holds most of the logic at the moment. Only logic held in the express server is randomization at the moment.
